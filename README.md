# 🎰 Python Slot Machine Game

A simple command-line slot machine game built with Python.

## Features

* Deposit money and manage balance
* Bet on up to 3 lines
* Random slot machine spins
* Win rewards for matching symbols
* Input validation

## How to Run

```bash
python main.py
```
